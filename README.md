# CSE 108 Lab 7

This project extends Lab 6 by adding a database using SQLAlchemy.

Instead of storing grades in memory, the application now uses a SQLite database (grades.db) to persist data.

The application supports full CRUD operations:
- View all students
- View a specific student
- Add a student
- Update a student
- Delete a student

The app is built with Flask and uses Flask-SQLAlchemy as the ORM.
